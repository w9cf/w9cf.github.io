Convert GR821A readings to admittance/impedance

Kevin Schmidt, W9CF

This applet converts the dial readings of a General Radio GR821A Twin-T impedance measuring circuit to admittance and impedance values incorporating residual corrections.

The input fields are the measurement frequency in MHz; the frequency scale of the conductance dial (either 1, 3, 10, or 30 MHz) used; the initial value of the capacitor; the final value of the capacitor, and the value read directly off the conductivity dial. The applet uses the default values from GR821A manual for the residual parameters. These can be changed by clicking Setup.
