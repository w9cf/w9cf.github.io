Phase-Shift Network Analysis and Optimization

Kevin Schmidt, W9CF
6510 S. Roosevelt St.
Tempe, Arizona 85283

Abstract:

The HA5WH phase shift network is analyzed. The ideal network with cyclic symmetry is described and simple design equations are given. Calculations that allow for component tolerances, show that well matched components must be used to obtain high quality results.
